研究について / About this project
本アプリは、日常の小さな行動提案とケア行動を通じて、キャラクターへの愛着と自己観察の関係を探る研究プロトタイプです。行動の回数・間隔、ケアの選択、再訪時の変化などをログ化し、CAPGAP（C・A・Pのギャップ）観点で分析します。教育・研究目的以外には利用しません。
